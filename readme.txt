A customizable sticky footer radio player supporting Shoutcast and Icecast streams.

== Description ==

Footer Radio Player is a lightweight and modern radio player for WordPress.

Features:

- Sticky footer player
- Shoutcast support
- Icecast support
- Live title via AJAX
- Scrollable radio text
- Dynamic Cover Art (iTunes API)
- Customizable colors
- Popup Player Generator
- Desktop-only volume control

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin
3. Configure it under Settings → Footer Radio Player
